# Pinpoint Customer Journey Map

Interactive visualization of the end-to-end customer journey for **Pinpoint SME** by PMAX.

## Live Demo

Open `index.html` in any browser, or enable GitHub Pages to view online.

## Features

- **15 steps** across 5 phases: Chuẩn bị → Tiếp cận → Proposal → Triển khai → Mở rộng
- **PIC filter**: BD, Account, Planner, Media
- **RACI matrix** for each step
- **Editable**: click any card to edit title, tasks, PIC, SLA
- **Add/remove** steps and tasks
- **Export/Import JSON** to save customizations
- **Card / List** view toggle
- **Print-friendly**

## Usage

1. Open `index.html` in browser
2. Click any step card to view details
3. Click "Chỉnh sửa" to edit
4. Use Export JSON to save your changes
5. Use Import JSON to restore
